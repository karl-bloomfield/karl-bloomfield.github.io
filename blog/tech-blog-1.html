What are the differences between relative,
absolute, and fixed positioning?

Research: freeCodeCamp example

Why learn the position property?
Well, you can make more interesting websites. I used a lot of Flexbox in this blog site, so I wanted to learn something new
and then put this on this page. With the position property, you can make an asymmetrical website
like the example in the right.

Flexbox and Grid limit placement around the X & Y axis. Your boxes will have exact measurements and placements.
Position property allows you to place content anywhere you wish.

There are 5 position property vales:

  1. relative
  2. absolute
  3. static
  4. fixed
  5. sticky

To move a box (<div>) we use 4 properties:

  1. top
  2. right
  3. left
  4. bottom

{position: static;}

this is the default value of every element.


{position: relative or absolute;}

they work the same way, except we use relative to identify the parent and use absolute to identify the child classes.


{position: fixed;}

this value will fix the position of your element on the screen, even when you scroll the browser.


{position: sticky;}

this value will fix the position of our element on the screen so it doesn't move.
